### Instalar projeto

Clonar repositório:
```bash
git clone https://github.com/vicenteprogammer/task-tracker.git
```
Instalar dependências:
```bash
npm i
```
Criar variáveis ambientes:
```bash
PORT=3000
```

Executar todo o código
```bash
npm start
```
